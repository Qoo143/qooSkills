# 週報管理 Skill (Weekly Report Management)

這個 Skill 用於標準化每週工作報告的撰寫與歸檔。報告將依據年份存放在專屬資料夾中，每個月使用一個 Markdown 檔案進行彙整。

## 目錄結構規範

報告必須存放在以下路徑：
- 路徑格式：`.claude/skills/weekly-report/<YYYY>/<MM>.md`
- 範例：`.claude/skills/weekly-report/2026/02.md`

## 報告格式規範

每一週的報告應包含以下區塊。如果某個區塊沒有內容，請保留標題並填入：

```markdown
# YYYY-MM-DD 工作進度
## 本週請假累積：
## 本週會議：
 - MM/DD - [會議名稱]
## 待處理 ：
## 處理中 ： 
 ### 需求單 ：
 ### 其他 ：
## 已完成 ：
 ### 需求單 ：
 ### 其他 :
## 上周建議修改事項 :
```

## 操作流程

1. **識別日期**：取得當前的系統日期。
2. **定位檔案**：
   - 確定該日期所屬的年份資料夾（例如 `2026`）和月份檔案（例如 `02.md`）。
   - 如果資料夾或檔案不存在，請自動建立。
3. **寫入內容**：
   - 如果檔案已存在，應在檔案**最上方**（或按日期倒序）插入新的週報區塊，以便快速閱覽。
   - 使用分隔線 `---` 區隔不同週次的報告。
4. **內容獲取**：
   - 詢問用戶本週的工作細節，或根據對話紀錄自動勾畫草稿。
